Introduction

Since I don’t use CSS frameworks in my front-end work, I needed a way to quickly build a header component without doing it from scratch for each new project. I don’t think I’m the only one who thinks in the same way.

As a start, I created 17 headers with slight differences between them. This is an ongoing project so expect more headers coming your way. If you have a suggestion, please use Github issues and I will be happy to implement it.
